# 💊 Medicine Management System

## 📌 Description
A Python desktop application that helps users manage medicines and get reminders on time.

## 🚀 Features
- Add medicines with time and dosage
- View all medicines
- Mark medicines as taken
- Delete medicines
- Desktop notifications for reminders
- Dashboard with statistics

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- SQLite (Database)
- Plyer (Notifications)
